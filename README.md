# DriverLens

A transparent, safety-first driver manager for Windows 10 and Windows 11. Inspect, verify, back up, and install drivers with confidence.

DriverLens is an open-source alternative to Snappy Driver Installer, built around a metadata-only GitHub driver index and a transparent WPF-UI desktop client for Windows.

Status: Phase 0 scaffolding

## Build

```powershell
dotnet build DriverLens.sln
```
